# Open WebUI community

NEURA Office ships as three separate community tools (one `.py` each).

## Tool pages

Install from the Open WebUI community (or paste from GitHub):

| Tool | GitHub |
|------|--------|
| Generate Documents | https://github.com/ianustec/openwebui-generate-documents |
| Generate Slides | https://github.com/ianustec/openwebui-generate-slides |
| Generate Spreadsheets | https://github.com/ianustec/openwebui-generate-spreadsheets |

Search the community for **Generate Documents**, **Generate Slides**,
**Generate Spreadsheets** (IANUSTEC).

## After a new release

The community listing has no release section. Do both:

1. Update the tool **description** (and the pasted `.py` / version).
2. Post a **comment** with the changelog + GitHub release link.

Template:

```
vX.Y.Z - Short title

What's new
- bullet
- bullet

GitHub: https://github.com/ianustec/openwebui-generate-…/releases/tag/vX.Y.Z

If you already installed the tool, replace the .py with the latest file from the repo.
```
